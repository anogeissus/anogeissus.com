--- layout: post title: "ApolloOne 4.7.0" date: 2026-07-22 17:05:00 +0000 summary: "Import files picked by ApolloOne GO, folder grouping in Contact Sheet, Ultra HDR support. Various enhancements and bug fixes." ---
  • NEW! Import files picked by ApolloOne GO — a new option in the memory card import copies only the RAW files you marked as Pick in ApolloOne GO on your iPhone or iPad (the green Finder tag), together with their matching JPEG / HEIF / TIFF files.
  • NEW! Folder grouping in Contact Sheet — when sorted by Path Name (or Custom Sort with Path Name as the first sort key), thumbnails are grouped by folder, each group headed by a Finder-style folder path bar that stays pinned while you scroll.
  • NEW! Ultra HDR support — exported HDR JPEGs now display in full HDR in Google Chrome, on Android devices, and in Google Photos, as well as on Apple devices.
  • Modernized In-App Purchase system — purchases, subscriptions, and restores have been migrated to Apple's StoreKit 2 for faster, more reliable purchase and restore handling.
  • Revamped help book — the help book has been revamped in all eleven languages; every option, menu, and button name quoted in the help now matches the app's UI wording exactly.
  • Fixed transparent PNG exports turning black when cropped or resized.
  • Fixed the color swatch being clipped at the bottom of the inspector.
  • The minimum system requirement is now macOS 12 Monterey.
  • Various enhancements and bug fixes.
