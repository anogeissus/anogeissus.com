Download ApolloOne User Script

ApolloOne makes use of a User Script to enable the Trash Put Back feature via Finder. It also provides you with the ability to synchronize file Comments between ApolloOne and the system. It is not absolutely necessary to install the script if you do not want these extra features.

Instruction for User Script installation

  1. Download the user script by clicking the Download Script button below.
  2. From ApolloOne's menu, select "Preferences… > Startup & Delete", and click the "Install" button.

We apologize for this inconvenience, this procedure is required by Apple in order for the app to be in compliant with the App Store standard.
